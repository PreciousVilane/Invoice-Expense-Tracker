#  Invoice & Expense Tracker

A full-stack web application for managing invoices, payments, and expenses, with a real-time financial dashboard.


## Features

###  Invoices
- Create invoices with multiple line items
- Auto-calculated totals
- Track invoice status (Pending / Paid)

### Payments
- Record payments against invoices
- Automatically mark invoices as paid
- Store payment history

###  Expenses
- Log business expenses
- Categorize and track spending

### Dashboard
- Total Income
- Total Expenses
- Profit calculation
- Outstanding invoices
- Data visualization (charts)


##  Tech Stack

### Backend
- Node.js
- Express.js
- SQLite

### Frontend
- React.js
- Recharts (for charts)
